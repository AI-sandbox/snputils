# Changelog

Release notes are published with [GitHub Releases](https://github.com/AI-sandbox/snputils/releases). Each tagged version on PyPI corresponds to a release entry there.

For the installed version in your environment:

```bash
snputils --version
# or
python -c "import snputils; print(snputils.__version__)"
```

Development builds between releases use setuptools-scm versioning from git tags; see the repository's default branch for unreleased changes.
